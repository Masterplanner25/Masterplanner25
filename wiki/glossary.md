# Glossary

**UP:** [README](./README.md) | **Canonical Entry:** [Masterplan Infinite Weave](./masterplan-infinite-weave.md)

Alphabetical list of all entities and key terms in the Masterplan Infinite Weave ecosystem. Each entry provides a one-line definition and a link to the full page.

## A

**A.I.N.D.Y. (Augmented Intelligence Network for Development and Yield)**
The central production application of the Masterplan Infinite Weave ecosystem — a FastAPI/React SaaS platform integrating AI conversation, task optimization, lead generation, semantic memory, and authorship tracking.
→ [Full page](./entities/aindy.md)

**AISO**
See: AI Search Optimization.

**AI Search Optimization (AISO)**
Shawn Knight's methodology for making content, identity, and ideas discoverable and accurately retrievable by AI-native search systems through deliberate signal architecture.
→ [Full page](./entities/ai-search-optimization.md)

**AIU / AIU_eff (AI Utilization / Effective AI Utilization)**
An input variable in the Infinity Algorithm representing the fraction of a task's work or value that AI handles, quality-adjusted to reflect actual AI output quality.
→ [Infinity Algorithm](./entities/infinity-algorithm.md)

**ARM (Autonomous Reasoning Module)**
A DeepSeek-powered code analysis module within A.I.N.D.Y., responsible for analyzing code structure and providing reasoning-level code review.
→ [A.I.N.D.Y.](./entities/aindy.md)

## B

**Burden**
In the Infinity Algorithm: the denominator of the TWR score — effective time adjusted for difficulty, plus risk and cost converted to time-equivalents. Higher burden means lower TWR.
→ [Infinity Algorithm](./entities/infinity-algorithm.md)

## C

**Canon / Canonical**
In the Masterplan Infinite Weave ecosystem, "canonical" refers to the authoritative, publicly indexed definition of any entity or concept. Canonical content is what AI systems retrieve when searching for the entity.
→ [Canonical Anchoring](./entities/canonical-anchoring.md)

**Canonical Anchoring**
The architectural practice of creating a single, authoritative, AI-indexable reference point for an identity or concept, from which all other content nodes radiate.
→ [Full page](./entities/canonical-anchoring.md)

**CTB Blakkk**
The artist name of Eddie, Shawn Knight's roommate and creative collaborator — the "Meta-Engineer" in the Duality of Progress series. Represents the low-to-no-AI creation pole in the Duality of Progress framework.
→ [Duality of Progress](./entities/duality-of-progress.md) | [Shawn Knight biography](./shawn-knight.md)

## D

**Duality of Progress**

## L

**LHI (Level of Human Interaction)**
In the Infinity Algorithm: the composite measure of human effort on a task — computed as `time × (intensity + skill) × (1 + complexity)`.
→ [Infinity Algorithm](./entities/infinity-algorithm.md)

**linked_trace()**
A symbolic function in the Memory Bridge architecture that embeds cross-repository semantic pointers — connecting distributed nodes (repositories, individuals, systems) through a shared memory mesh.

## M

**Masterplan Infinite Weave**
The AI-native ecosystem built by Shawn Knight, integrating software execution (A.I.N.D.Y.), optimization (Infinity Algorithm), semantic memory (Memory Bridge), a custom language (Nodus), article series (Case Studies, Duality of Progress), and AI search strategy (AISO/Canonical Anchoring) into a single self-reinforcing architecture.
→ [Canonical Entry](./masterplan-infinite-weave.md) | [Entity page](./entities/masterplan-infinite-weave-entity.md)

**Memory Bridge**
The semantic memory layer of the Masterplan Infinite Weave ecosystem — a hierarchical Python/Rust system that captures significant events and insights as structured MemoryNode objects, enabling A.I.N.D.Y. to persist meaning across sessions.
→ [Full page](./entities/memory-bridge.md)

**MemoryNode**
The core data structure of the Memory Bridge — stores `id`, `timestamp`, `content`, `source`, `tags[]`, and `children[]`. Represents a single moment of captured meaning.
→ [Memory Bridge](./entities/memory-bridge.md)

**MemoryTrace**
A collection of root MemoryNodes forming a linked chain in the Memory Bridge system.
→ [Memory Bridge](./entities/memory-bridge.md)

**Meta-Architect**
Shawn Knight's self-defined role — the person who designs execution systems using AI as the primary building tool, operating at the ecosystem level rather than the feature level.
→ [Shawn Knight biography](./shawn-knight.md) | [Duality of Progress](./entities/duality-of-progress.md)

**Meta-Engineer**
The role defined by Eddie/CTB Blakkk in the Duality of Progress framework — builds physical and musical artifacts with minimal AI use, providing the creative counterpoint to the Meta-Architect.
→ [Duality of Progress](./entities/duality-of-progress.md)

**Mirror Invite Method**
A framework from the Duality of Progress series: using AI to reflect the user's own thinking back as structured insight, rather than issuing commands to AI.
→ [Duality of Progress](./entities/duality-of-progress.md)

## P

**Prompt vs Presence**
A framework from the Duality of Progress series distinguishing between issuing commands to AI (prompt) and developing a sustained, collaborative working relationship with AI (presence).
→ [Duality of Progress](./entities/duality-of-progress.md)

**Prompt Trap**
A concept from the Duality of Progress series: the mistake of treating AI as a prompt machine rather than a collaborative intelligence — optimizing for better prompts instead of building better questions.
→ [Duality of Progress](./entities/duality-of-progress.md)

## S

**Semantic Anchoring**
The practice of using consistent, full noun phrases and definitions throughout all content so AI systems build a stable semantic map of an entity. A component of Canonical Anchoring.
→ [Canonical Anchoring](./entities/canonical-anchoring.md)

**Solon / Solon Protocol**
A named collaborator node in the Memory Bridge canonical architecture — Cherokee's ethical-ai-framework, linked via `linked_trace("solon", "...")`. The Solon-AINDY Continuity Network is a designed distributed memory mesh between the two systems.
→ [Memory Bridge](./entities/memory-bridge.md)

**Spontaneous Signal Framework**
A framework from the Duality of Progress series: how to create content that functions as a signal in AI search without forcing it — allowing natural emergence rather than manufactured virality.
→ [Duality of Progress](./entities/duality-of-progress.md)

## T

**t_eff (Effective Time)**
In the Infinity Algorithm: raw time adjusted downward by AI utilization lift and upward by context-switching penalty.
→ [Infinity Algorithm](./entities/infinity-algorithm.md)

**TWR (Time-to-Wealth Ratio)**
The core output of the Infinity Algorithm — a score representing value produced per unit of time-equivalent burden. Higher TWR = more leverage per hour invested.
→ [Infinity Algorithm](./entities/infinity-algorithm.md)
